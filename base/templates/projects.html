PROJECT 1. CNN 모델을 이용한 숫자 인식 프로젝트

숫자 이미지를 인식하는 CNN 모델 입니다. 손으로 쓴 숫자 이미지 6만개로 사전학습한 CNN 모델을 활용하여 업로드한 이미지에서 숫자를 인식하고 분류하는 프로젝트 입니다. 만개의 데이터로 검증한 결과 분류 정확도는 97.69% 입니다.

PROJECT 2. CNN 모델을 활용한 의류 분류 프로젝트

10개의 패션아이템을 분류하는 CNN 모델 입니다. Fashion MNIST 데이터셋(28X28 흑백이미지) 6만개로 학습시켰고, 분류할 수 있는 의류는 다음과 같습니다.
T-shirt/top, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot. 만개의 데이터로 검증한 결과 분류 정확도는 88.86% 입니다.

PROJECT 3. VGG 모델을 활용한 사물 분류 프로젝트

10개의 사물을 분류하는 VGG 모델 입니다. CIFAR-10 데이터셋(32x32 크기의 컬러 이미지) 5만개로 학습시켰고, 분류할 수 있는 사물은 다음과 같습니다.
비행기, 자동차, 새, 고양이, 사슴, 개, 개구리, 말, 배, 트럭. 만개의 데이터로 검증한 결과 분류 정확도는 80.31% 입니다.
